# Contributing

欢迎提交 issue 或 pull request 来补充论文、代码入口、绘图模板和复刻经验。

## 推荐补充格式

新增论文时，请尽量提供：

- 论文题目和 DOI / 期刊页面
- GitHub、Zenodo 或官方代码入口
- 可复刻的 figure / panel
- 使用的 R / Python 包
- 适合沉淀成的绘图 skill 模块
- 证据等级：A 表示已有明确 figure、notebook、source data 或 reproduce 入口；B 表示已有代码仓库但需要进一步定位绘图脚本

## 边界

本项目聚焦可视化学习和绘图工作流，不评价论文结论强弱，也不提供医学诊疗建议。
