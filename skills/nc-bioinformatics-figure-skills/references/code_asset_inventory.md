## Use Rules

- Prefer `A1` repositories for direct code replication.
- Use `A2` repositories to learn project structure, package choices, and panel logic, then verify exact figure paths before copying.
- Keep `A3` repositories as concept references until figure/source-data paths are found.
- Do not copy biological claims from any repository; copy input schema, plotting structure, export conventions, and panel hierarchy only.
