## Practice Ladders

### Trajectory

1. Start with MoFlow `plot_cell.py` and `plot_gene.py` concepts.
2. Reproduce a toy embedding with arrows and a gene trend panel.
3. Add ddHodge-style vector-field notebook as advanced layout.
4. Use ILC-NK-HIPEC figure scripts to learn immune trajectory panel hierarchy.

### Spatial Niche

1. Start with scHolography `R/Visualization.R` and `neighborPlots.R`.
2. Use Sopa/FineST to learn image or tissue overlay.
3. Use Tumor-cell-villages to learn niche/state/network biological panels.
4. Transfer to project as map + quantification + marker/communication evidence.

### Communication

1. Start with FastCCC report scripts.
2. Build LR heatmap before network.
3. Add Human_endometrium or FineST for NicheNet/CellPhoneDB/spatial validation.
4. Keep full LR tables in supplement and filtered networks in main figures.

### Benchmark

1. Use NEP_comparison and cPeaks for metric/rank plotting.
2. Use CellANOVA/SMART for integration benchmark structure.
3. Pair every metric matrix with one interpretable biological example.
